Estandares

El campo que muestre un * indica que es un obligatorio y se debe intruducir el valor para poder guardar la informacion.

 

En las tablas la columna obligatoria se hace notar por el color rojo del borde.

 

Este campo realiza busqueda de informacion, conocido como codLookup (Selector de Codigo).

Es posible escribir el codigo en el cuadro de texto para una seleccion rapida. (Opcional)

El boton (...) se utiliza para abrir la ventana de seleccion. Tecla F1

El boton (+) se utiliza para abrir la ventana de edicion o creacion. (Opcional) Tecla F2

El boton (O) se utliza para abrir la venta compleja de visualizaci{on y edicion de registro. (Opcional)

 

El campo fecha requie 2 digitos para el día, 2 digitos para el mes y 4 digitos para el año.

La tecla F1 llena el campo con la fecha actual.

 ó la tecla F2 abre la ventana de seleccion de fecha.

 

En las tablas ó grids:

Al hacer clic sobre la cabecera de la columna la ordena.

Al hacer clic derecho ó CTRL + Q sebre el contenido de la columna da la posibilidad de filtrar.

Donde el criterio sea: igual, contiene, comienza con, termina con. En las fechas y numeros por un rango.

Se puden ocultar y visualizar coluimnas espesificas.

El sub-menu de Gestión del perfil del grid (Opcional) permite guardar la configuracion de un grid, el filtro aplicado, las columnas visibles, las columnas ordenadas.

 

Botones:

 

A traves de la aplicacion conseguiras repetidamente los anteriores botones para un uso mas intuitivo, siempre cumplen la misma funcionalidad:

 

  1. Agregar
  2. Eliminar
  3. Editar
  4. Guardar
  5. Actualizar
  6. Exportar
  7. Filtrar
  8. Enviar e-Mail
  9. Barra de Navegación

 

1.  Agregar - CTRL + I

    Para agregar un nuevo registro al renglon el cual este asociado. Ejemplo si esta en personas agrega una nueva persona.

2.  Eliminar - CTRL + D

    Para eliminar un nuevo registro al renglon el cual este asociado. Ejemplo si esta en personas elimina una nueva persona.

3.  Editar - CTRL + E

    Antes de modificar algún campo para que permita el cambio luego se guarda o se actualiza para cancelar.

4.  Guardar - CTRL + S

 

    Guarda la información ya sea nueva o después de alguna edición.

5.  Actualizar - CTRL + Z

     Refresca la vista y cancela los cambios que no quieren ser guardados.

6.  Exportar - CTRL + X

     Exportarla en un archivo o directamente en los formatos (XLS, CSV, XML, HTML, PDF, RTF).

7.  Filtrar - CTRL + F

     Filtra la data que se está mostrando generalmente usado en listados y para hacer búsquedas más específicas.

8.  Enviar e-Mail 

    Para el envio de correo electronico usa el boton E-Mail

10. Barra de Navegación

    Se utiliza para navegar a través de los registros de la tabla
Esta compuesta por:

  •    Ir al registro anterior
  •    Ir a la página anterior
  •    Ir al primer registro
  •    Ir al siguiente registro
  •    Ir a la siguiente página
  •   Ir al ultimo registro
  • Ir a una pagina en espesifico
