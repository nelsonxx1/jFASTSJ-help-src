ABRIR CALENDARIO
 Comando de teclado usado para abrir el calendario popup en los campos y columnas de fecha
TECLA = F2

FECHA ACTUAL CALENDARIO
 Comando de teclado usado para establecer la fecha actual en los campos y columnas de fecha
TECLA = F1

ABRIR LOOKUP
 Comando de teclado usado para abrir la ventana de selección en los campos y columnas lookup
TECLA = F1

EDITAR LOOKUP
 Comando de teclado usado para abrir la ventana de edición en los campos y columnas lookup
TECLA = F2

BOTON INSERTAR
 Comando de teclado usado para ejecutar el boton agregar
TECLA = CTRL+I

BOTON EDITAR
 Comando de teclado usado para ejecutar el boton editar
TECLA = CTRL+E

BOTON GUARDAR
 Comando de teclado usado para ejecutar el boton guardar
TECLA = CTRL+S

BOTON ACTUALIZAR
 Comando de teclado usado para ejecutar el boton actualizar
TECLA = CTRL+Z

BOTON ELIMINAR
 Comando de teclado usado para ejecutar el boton eliminar
TECLA = CTRL+D

BOTON EXPORTAR
 Comando de teclado usado para ejecutar el boton exportar
TECLA = CTRL+X

BOTON FILTRAR
 Comando de teclado usado para ejecutar el boton filtrar
TECLA = CTRL+F

MENU DE TABLA
 Comando de teclado usado para abrir el menu popup en cada columna de las tablas
TECLA = CTRL+Q

MENU DE ARBOL
 Comando de teclado usado para posicionar el foco en el buscador de funciones del Menu
TECLA = F3

IR A FUNCION
 Comando de teclado usado para posicionarse en la función seleccionada en el buscador de funciones del menu
TECLA = TAB

REMOVER FILTROS
 Comando de teclado usado para remover un filtro en el panel de filtrado rapido
TECLA = CTRL+R

AGREGAR FILA
 Comando de teclado usado para agregar una nueva fila en una tabla.(En modo inserción o edición)
TECLA = FLECHA ABAJO

REMOVER FILA
 Comando de teclado usado para eliminar una fila extra ya agregada a una tabla.(En modo inserción o edición)
TECLA = FLECHA ARRIBA
